# langchain-AI-generate-interview-question

### How to run? 

1. Create an environment

```bash
conda create -n interview python=3.10 -y

conda activate interview
```

2. Install requirements

```bash
pip install -r requirements.txt
```

# Note
Project's still under progress since it's currently not possible to create an answer knowledge data based on just the JD. Therefore, only the questions generated by this can be count as something valuable.
